Turn one run into

Pull a run straight from Strava, drop in a Dexcom Clarity export, and get a single beautiful report showing how your glucose moved before, during, and after the run.

1
Connect Strava
One-time setup (~2 min): create your personal Strava app
  1. Go to strava.com/settings/api (logged in as the runner) and create an app — any name and website work.
  2. Set Authorization Callback Domain to this site's domain.
  3. Copy the Client ID and Client Secret into the boxes below.
  4. Hit connect — Strava will ask you to approve read-only access, then send you straight back here. Nothing is stored; you stay connected only for this visit.
Already have an access token with activity:read scope?
2
Add your glucose data
How to export from Dexcom Clarity (~30 sec)
  1. Open the Dexcom Clarity app or clarity.dexcom.com.
  2. Go to reports, and look for Export / Download CSV (usually top-right or in the menu).
  3. Export a range that covers the day of the run.
  4. Upload that CSV file below — nothing is uploaded anywhere, it's read in the browser only.
