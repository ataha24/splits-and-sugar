Pull a run straight from Strava, drop in a Dexcom Clarity export, and get a single fun report showing how her glucose moved while she ran.

1
Connect Strava
How to get a Strava access token (one-time, ~2 min)
  1. Go to strava.com/settings/api (log in as the runner).
  2. Fill in any app name and website (e.g. "My Run Reports" / "https://example.com") and create the app.
  3. On the page that appears, copy the value next to "Your Access Token".
  4. Paste it below. It only lets this page read her own activities.
2
Add her glucose data
How to export from Dexcom Clarity (~30 sec)
  1. Open the Dexcom Clarity app or clarity.dexcom.com.
  2. Go to reports, and look for Export / Download CSV (usually top-right or in the menu).
  3. Export a range that covers the day of the run.
  4. Upload that CSV file below — nothing is uploaded anywhere, it's read in the browser only.
